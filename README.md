# Instagram Social media Analysis (SQL Project)

## About this project
In this project, I analyzed Instagram user data to understand how users interact on the platform.

The goal was to study user activity, engagement patterns, hashtags performance, and identify potential influencers.

---

## What I did
- Explored multiple tables like users, photos, likes, comments, and follows
- Used SQL queries to analyze user behaviour and engagement
- Calculated metrics like engagement rate, activity levels, and hashtag performance
- Identified active users, inactive users, and top influencers

---

## Key findings
- Only 13% of users are highly active, while 33% are inactive  
- 23% of users never liked any post  
- Users with more posts get higher engagement  
- Hashtags like #dreamy, #beauty perform better  
- Very few users (top influencers) drive most engagement  

---

## Insights
- Platform has many passive users  
- Engagement is concentrated among few users  
- Hashtags play an important role in visibility  
- Content frequency directly affects engagement  

---

## Suggestions
- Re-engage inactive users using notifications  
- Reward top creators with incentives  
- Use high-performing hashtags in campaigns  
- Encourage users to post more content
